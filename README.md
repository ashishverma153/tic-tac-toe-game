# Tic-Tac-Toe Game

An interactive, responsive Tic-Tac-Toe game built using vanilla modern web technologies, featuring clean DOM manipulation, event handling, and real-time state tracking.

## 🛠️ Tech Stack
- **HTML5:** Semantic structural layout for the game grid and interface components.
- **CSS3:** Modern flex/grid layout configurations with dynamic hover animations, vibrant active states, and custom styling.
- **Vanilla JavaScript:** Event-driven logic managing player turn states, grid validation matrices, reset states, and win/draw detection conditions.

## 🚀 Future Enhancements
- Integrate a Minimax algorithm AI opponent for an unbeatable single-player mode.
- Add local scoreboard persistence using browser LocalStorage tracking.
